# Vehicle-Showroom
A command line system for vehicle showroom using Java.<br/>
**Instructions**<br/>

1.Clone the repo:$ git clone https://github.com/merajatul/Vehicle-Showroom.git<br/>
2.Install: Make sure you have java installed<br/>
3.Launch: $ cd \src<br/>
          $ javac Main.java<br/>
          $ java Main
